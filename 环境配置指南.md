conda activate csdi
## Python 版本推荐

### 推荐版本：**Python 3.8** ⭐

**理由：**
- ✅ 2021年项目的最佳兼容版本
- ✅ PyTorch 1.x 完全支持
- ✅ 所有依赖库兼容性最好
- ✅ 稳定可靠，广泛使用

### 备选版本：
- **Python 3.7**：最低支持版本（如果3.8不可用）
- **Python 3.9**：也可以使用，但需要确保PyTorch版本兼容

### 不推荐：
- ❌ Python 3.10+：可能某些依赖库不兼容
- ❌ Python 3.6及以下：已不支持

---

## 环境配置方法

### 方法一：使用 Conda（推荐）⭐✅

```bash
# 1. 创建conda环境（Python 3.8）
conda create -n csdi python=3.8 -y

# 2. 激活环境
conda activate csdi

# 3. 安装PyTorch（根据你的CUDA版本选择）
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126
显示已经有PyTorch版本: 2.3.0+cu118；绑定的CUDA版本: 11.8，因为新建的csdi虚拟环境并非 “完全纯净”，而是继承 / 复用了base环境中已安装的 PyTorch，是 Conda 的正常优化，且csdi环境的 torch 是独立的（和 base 隔离），不影响后续使用。只要验证torch.cuda.is_available()为True，就能正常用 GPU 跑代码，无需处理～
# 4. 安装其他依赖
pip install pandas numpy tqdm pyyaml matplotlib wget

# 5. 安装linear_attention_transformer
pip install linear-attention-transformer

# 6. 安装requests（download.py需要）
pip install requests
```
运行`python test_environment.py`文件来检查是否安装全部包。
